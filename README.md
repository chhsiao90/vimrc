## Install
### Install Vundle
```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

### Install vimrc
```
git clone https://github.com/chhsiao90/vimrc.git ~/.vim_runtime
sh ~/.vim_runtime/install.sh
```
